Project Title: Power BI Churn Analytics Project

Overview: Designed and developed an interactive Power BI dashboard that provides insights into customer churn, revenue trends, and business performance. The project demonstrates data modeling with star schema, Power Query transformations, advanced DAX calculations, Row-Level Security (RLS), drillthrough, bookmarks, and executive reporting.

Tools Used: 
- Power BI Desktop
- Power Query
- DAX
- Star Schema
- Row-Level Security (RLS)
- Bookmarks
- Drillthrough


Goal: Design and develop an end-to-end Power BI dashboard that delivers meaningful business insights using data modeling, DAX, and interactive reporting.
